Dodatki dostarczone z Blenderem

  • Export Paper Model - prosty model (nawet oteksturowany) można rozłożyć na 2D i wydrukować na drukarce do złożenia
  • Import AutoCAD DXF Format
  • Import Images as Planes - proste dodawanie obrazków w 3d, wiele na raz.
  • Copy Attributes Menu
  • Modifier Tools
  • Sun Position - rzeczywiste ustawienie kąta padania słońca i animacji słońca
  • Material Utilities - SHIFT+Q - świetne narzędzie do podmieniania materiałów
  • Bsurfaces GPL Edition - narzędzie do modelowania
  • F2 - narzedzie do modelowania
  • LoopTools - narzędzie do modelowania
  • tinyCad Mesh tools - super narzedzie jak chcesz przedłużyć linię, bądź połączyć narożniki, bądź znaleźć punkt przecięcia
  • Node Arrange - pomocne przy pracy z nodami
  • Node Presets - pomocne przy pracy z nodami
  • Node Wrangler - pomocne przy pracy z nodami
  • Align Tools - wyrównywanie elementów w 3D
  • Bool Tool - wycinanie obiektów obiektami
  • Carver - rzeźbienie wycinaniem "nożem"
  • Edit Linked Library - łatwe otwieranie plików zalinkowanych
  • Freestyle SVG Exporter - Jak używasz generowania linii na rysunku to można je wyeksportować wektorowo

Dodatki które warto zainstalować

  • BlenderBIM - kombajn do pracy z danymi BIM, od 3D do zestawień, kosztorysów, generowania dokumentacji 2D.
  • BlenderGIS - kombajn do geolokalizacji i importowania danych.
  • Modular Tree - darmowe Generowanie niesamowitych drzew!
  • PRJ - taki prosty generator rzutów z modelu 3D (nie testowałem)
  • CAD Transform - przenoszenie obiektów tak jak w revicie Wideo instruktażowe
  • CAD Sketcher - bardzo cadowskie generowanie brył za pomocą constraits.

Dodatki płatne

  • https://www.scatterforblender.com/

Assety

  • Poly Haven + ADDON priorytetowa, wszystko na licencji CC0!, utrzymuje się z datków, polecam kupić addon aby ich wesprzeć
  • BlendSwap - modele 3D funkcjonuje od dawna
  • BlenderKit + ADDON - modele, textury, hdri
  • Sketchfab + ADDON - modele 3d
  • Quixel Megascans - darmowe tylko w przypadku Unreal Engine, wysokiej jakości skany i tekstury 19$/miesiąc, jednego miesiąca można ściągnąć 45 materiałów, lub 22 modele 3d, przemyśleć co się chce posprawdzać za darmo jak działa i potem zakupić przez 1 miesiąc.
  • textures.com - płatne, ale warto się zastanowić, wysokiej jakości textury, i skany 3d
  • https://ambientcg.com/
